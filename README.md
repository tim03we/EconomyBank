# EconomyBank

EconomyBank is a bank plugin that allows you to save your money in the bank.

# Commands
Command | Sub Command | Description
------- | ----------- | -----------
/bank | ~ | Help List
~ | add | Deposit money
~ | remove | Withdraw money
~ | show | See your account balance
~ | see | See the account balance of others
